# TV Script Generator

This project demonstrates the use of RNNs for script generation. It will create a new "fake" TV script based on the 9 seasons of Seinfeld TV serries. 

## Installation

Clone this repo:
```
git clone https://github.com/puhach/tv-script-gen.git
```
